Classroom / Curriculum SERVICE

STUDY PLACE.meの教習内容・分野、サービスなど。

コース内容

  • Web site construction Web制作コース

    Webサイトの制作方法について学習します。 基本のHTML/CSS/JavaScriptのコーディングを学びながら、インターネットの基礎知識、検索エンジンの仕組み、ネットーワーク・セキュリティなど、Webサイト制作に必要な知識も習得。またWebサイトの航海に必要なサーバの知識やドメインの仕組みなど、制作物を公開するまでの手順もマスターします。

  • Web designing Webデザインコース

    デザインヒアリングの手法をはじめ、パソコンやインターネットに依存しないクリエイティブなコミュニケーション力を身につけながら、Photoshopなど専門のソフトウェアを用いたサイト設計や、イメージ制作に必要なデザインスキルを習得。またWebサイト以外のシチュエーションにも応用できる、デザインの基礎・概論をマスターしていきます。

  • Web marketing Webマーケティングコース

    シェアNo.1の解析ツール「Google アナリティクス」を使用し、Webサイトへの訪問数や効果測定、ターゲットの絞り込みなど、データに基づいたマーケティング手法を学習。アクセス解析やサイト分析、競合分析、戦略立案といった、集客のための方法や対策をロジカルに考えていきます。

  • Contents writing コンテンツライティングコース

    読み手の心に響くキャッチコピーの作り方や、読み進めてもらうための見出しの付け方、相手にきちんと伝わる文章の書き方、検索エンジンに評価されるための対策キーワードを意識した記事作成のテクニックをマスター。サイト制作やパンフレット作成など、Webから紙媒体まで幅広く対応できるライティングスキルを身に付けます。

  • Web site construction using CMS CMSビルドコース

    Webサイトを管理・更新するためのシステム「CMS」をインストールする段階からプログラミング、そして公開まで、ブログや企業サイトなどのコンテンツ制作に必要な一連の流れを学びます。本コースでは、主にWordPressを使ったコンテンツ制作を行うための構築方法を学習していきます。
